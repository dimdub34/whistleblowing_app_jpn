グループの構成はパート1と同じです。

コンピュータプログラムは、この部屋にいるすべてのグループの中からランダムに1つのグループを選びます。 このグループはパッシブグループと呼ばれます。 残りのグループはアクティブグループと呼ばれます。

パッシブグループのメンバーの最終的な個人報酬は、パート1での報酬と、パート2でアクティブグループのメンバーが行う決定によって決まります。 パッシブグループのメンバーはこのパートで意思決定を行わないため、以下の説明はアクティブグループのメンバー向けです。
全員が説明を読み終えると、あなたがパッシブグループなのか、アクティブグループなのかが通知されます。

アクティブグループにおける役割と決定

各アクティブグループでは、コンピュータプログラムがプレイヤーにランダムに役割を割り当てます。 グループにはレッドプレイヤー1名とブループレイヤー2名がいます。

レッドプレイヤーは、パッシブグループからECUを盗むことで自身の報酬を増やす選択肢を持ちます。 具体的には、レッドプレイヤーは「盗む」か「盗まない」かの二択を行います。

仕組みは次のとおりです：
レッドプレイヤーがECUを盗むと、レッドプレイヤーは{{ STEALING_AMOUNT }} ECU を獲得し、 パッシブグループは{{ STEALING_LOSS }} ECU を失います。 これは、パッシブグループの各メンバーが {{ STEALING_LOSS_INDIV }} ECU ずつ失うことを意味します（{{ STEALING_LOSS_INDIV }} ECU × {{ PLAYERS_PER_GROUP }} = {{ STEALING_LOSS }} ECU）。 レッドプレイヤーが盗まない場合、何も変化せず、全員の報酬はパート1のままです。

盗み行為には{{ STEALING_PENALTY }} ECU の罰則があります。 ただし、レッドプレイヤーが罰則を受けるのはブループレイヤーの1人に報告された場合のみで、報告されても必ず罰則が適用されるわけではありません。

報告のルール

各アクティブグループのブループレイヤーは、レッドプレイヤーがパッシブグループからECUを盗んだ場合に報告するかどうかを決めます。 レッドプレイヤーが盗まなかった場合、報告は行われません。 報告は匿名で行われ、実験者に送られます。

ブループレイヤーがレッドプレイヤーを報告し、レッドプレイヤーがECUを盗んでいた場合、 ブループレイヤーは{{ REPORTING_COST }} ECU の報告コストを支払います（パート1の報酬から差し引かれます）。

  • レッドプレイヤーがECUを盗んだ場合：
    • 報告したブループレイヤーは{{ REPORTING_COST }} ECU の報告コストを負担します。
    • 報告されたレッドプレイヤーは、{{ AUDIT_PROBABILITY_STR }} の確率で {{ STEALING_PENALTY }} ECU の罰則を受けます（パート1の報酬と盗んだ {{ STEALING_AMOUNT }} ECU から差し引かれます）。 {{ NON_AUDIT_PROBABILITY_STR }} の確率で罰則はありません。
      • {{ if subsession.reward }}
    • レッドプレイヤーが罰則を受けた場合、ブループレイヤーは{{ REPORTING_REWARD }} ECU の報酬を受け取ります。
      • {{ endif }}
  • レッドプレイヤーがECUを盗まなかった場合：
    • 報告は行われません。
    • レッドプレイヤーに罰則はありません。
報告のランダム適用

レッドプレイヤーがECUを盗んだ場合、コンピュータはブループレイヤーの決定のうち1つをランダムに選びます。

  • 選ばれた決定が「報告する」の場合、ブループレイヤーは報告コストを負担し、レッドプレイヤーは報告されます。
  • 選ばれた決定が「報告しない」の場合、レッドプレイヤーは報告されません。

選ばれなかったブループレイヤーの決定は影響しません。
レッドプレイヤーが盗まなかった場合、何も起こりません。

重要ポイントのまとめ

1つのグループがランダムにパッシブグループとして選ばれます。 その他のグループはアクティブグループです。
各アクティブグループでは：

  • 1名がレッドプレイヤー、2名がブループレイヤーです。
  • レッドプレイヤーはパッシブグループからECUを盗むかどうかを決めます （盗むと {{ STEALING_AMOUNT }} ECU を獲得し、パッシブグループは {{ STEALING_LOSS }} ECU を失います）。
  • ブループレイヤーは、レッドプレイヤーが盗んだ場合に報告するかどうかを決めます。
  • レッドプレイヤーが盗んだ場合：
    • コンピュータがブループレイヤーの決定のうち1つをランダムに選びます。
    • 選ばれた決定が「報告する」の場合、ブループレイヤーは {{ REPORTING_COST }} ECU を負担し、 レッドプレイヤーは {{ AUDIT_PROBABILITY_STR }} の確率で {{ STEALING_PENALTY }} ECU の罰則を受けます。 {{ NON_AUDIT_PROBABILITY_STR }} の確率で罰則はありません。 {{ if subsession.reward }} レッドプレイヤーが罰則を受けた場合、ブループレイヤーは {{ REPORTING_REWARD }} ECU を受け取ります。 {{ endif }}
    • 選ばれた決定が「報告しない」の場合、レッドプレイヤーは報告されません。
  • レッドプレイヤーが盗まなかった場合、何も起こりません（報告なし）。
